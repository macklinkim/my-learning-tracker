# 개발 진행 과정 (Development Phases)

## Phase 4: 고급 상호작용 (Interactive Features)

**목표:** 사용자 경험(UX)을 향상시키기 위한 인터랙티브 기능 및 고급 상태 관리를 도입합니다.

### 상세 작업 내용
- Zustand를 활용하여 필터링 상태, 모달 열림/닫힘 등의 전역 UI 상태 관리 구현.
- dnd-kit을 도입하여 학습 아이템을 '할 일 / 진행 중 / 완료' 상태로 드래그 앤 드롭 이동시키는 칸반 보드 구축.
- React Hook Form과 Zod를 사용한 복잡한 데이터 입력 모달(새 학습 항목 추가 등) 구현.
